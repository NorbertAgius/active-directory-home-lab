\# Active Directory Home Lab



This project documents a beginner-friendly Windows Server Active Directory home lab.



\## Project Goal



The goal of this lab is to understand how Active Directory is used in real IT environments to manage users, computers, groups, permissions, and security policies.



\## Technologies Used



\- Windows Server

\- Active Directory Domain Services

\- Group Policy

\- DNS

\- VirtualBox / VMware

\- Windows Client VM



\## Lab Plan



1\. Install Windows Server in a virtual machine

2\. Install Active Directory Domain Services

3\. Promote the server to a Domain Controller

4\. Create users and security groups

5\. Join a Windows client machine to the domain

6\. Configure shared folders and permissions

7\. Apply basic Group Policy settings
